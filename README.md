
# 说明

这是闲耘私人的博客，主要用于发表个人的观点；知识、信息积累与分享。

## 目录结构

```
/                         博客首页
/:year                    博客按年份归档页
/tags                     博客按标签归档页
/post/:title.html         博文详情页
/search                   搜索首页
/404                      公共 404 页面
```
