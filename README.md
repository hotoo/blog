
# 说明

这是闲耘私人的博客，主要用于发表个人的观点；知识、信息积累与分享。

## 目录结构

```
/post/:title.html
/archives/:year           指定年份的存档列表
/archives/:year/:month    指定月份的存档列表
/tags                     标签云列表
/tag/:tag-name            特定标签的博文列表
/categorys                分类名列表
/category/:category-name  特定分类的博文列表
/search                   搜索首页
/projects
/twitter
/movies
/musices
/404
```
