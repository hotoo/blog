Vimwiki 介绍

BUG

  • 不支持嵌套标签，如： 
    = [link desc] =
  • v0.9.9 及其之前的版本中，表格对中文支持不佳，1.0 Beta 之后有较大修正。 
    let g:vimwiki_use_mouse = 1
let g:vimwiki_camel_case = 0
let g:vimwiki_CJK_length = 1
  • GTD (/TODO) list 使用 <C-Space> 切换 List Item 的 check 状态，但这个 快捷键是中文输入法的默认快捷键。
