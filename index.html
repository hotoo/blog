转换 Vimwiki 为 Markdown ¶

sed -f ex -i *.md

README

相对 Wiki 语法，和 Wiki 对应工具支持情况，我更加喜欢 Markdown 了。 本地配合使用 Vimwiki 来编辑查看，发布使用 nico 来转换，而且有 Github 的原生支持， 使用 Markdown 是不二之选。

为了减少转换工作量，我简要写了这个脚本，可以初步将 vimwiki 语法的项目， 自动转换成 markdown 语法。
