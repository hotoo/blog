『已结束』庆祝 Vim 7.4 发布，发起团购 Tee + 团捐活动(2013)

团购活动已经结束，但你仍然可以自行印制或购买（有人抱怨质量不佳）
团捐活动已经结束，但你仍然可以 自行捐助

以 @vimcn 的身份捐赠 159欧元，实际人民币 1377.94元。 参考团捐的同学请查收邮件并于 2013年10月份之前行使您的 Vim 新特性赞同或反对票。

于 2013年08月10日发发布的 Vim 7.4 的主要特性更新如下：

  1. 增强 Python 支持；
  2. 内置更快的正则表达式引擎；
  3. bugs fix.

为了庆祝这一重大的事件，发起这次团购 VimTee + 团队捐助的活动。 活动截至 2013年08月15日，请有兴趣的同学火速参与。
